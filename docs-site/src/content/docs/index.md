# dumling docs

`dumling` is a TypeScript and Zod package for learner-facing linguistic annotation centered on hydrated `lemma`, `surface`, and `selection` DTOs.

These docs are authored in Markdown, rendered to static HTML by Astro, and emitted as static Markdown siblings for agents and other text-first consumers.

## Entrypoints

| Import path | Purpose |
| --- | --- |
| `dumling` | Root runtime API |
| `dumling/types` | Public DTOs and type helpers |
| `dumling/schema` | Runtime Zod schemas |
